인코딩이 깨진 상태라 완벽 복원은 안 되지만, 문맥으로 최대한 재구성해봤어:

1. CAN 인터페이스 활성화 (최초 1회 또는 재부팅 때마다, sudo 필요)

sudo ip link set can0 up type can bitrate 1000000

2. 빌드

cd /home/leejaejun/Documents/can_ak45
make

3. 실행

./ak45_ctrl              # 인자 없이 시작 (모니터링만, 실행 중 각도 입력 가능)
./ak45_ctrl 30            # 초기 목표 위치 30도로 시작
./ak45_ctrl -45           # 음수 각도도 가능

실행 중 화면에 "> 목표각도 입력(도, q=종료):" 프롬프트가 뜨면 새 각도를 입력 후 Enter →
새 창을 열 필요 없이 계속 실행된 채로 목표 위치가 즉시 갱신됩니다.
피드백(위치/속도/전류/온도/에러)은 실시간으로 계속 출력됩니다.
종료: Ctrl+C 또는 프롬프트에 q(또는 quit) 입력 → 안전 정지(Current Brake 0A) 후 종료.
처음 테스트할 땐 작은 각도(10~15도)부터 시작 권장.

(선택) 다른 터미널에서 실제 CAN 프레임 확인

candump can0
